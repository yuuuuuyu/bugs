# 表单多个下拉级联的逻辑事件
:::tip
下拉框的数据源配置改为通过逻辑事件配置后，实现了表单的多个下拉级联处理。
:::

### 业务场景

客户、项目、服务三个下级联选择，选择完客户后再加载项目的下拉数据，选择完项目后再加载服务的下拉数据
