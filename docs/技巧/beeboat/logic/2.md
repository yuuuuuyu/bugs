# 【逻辑事件系列】判断表格选中数据来控制按钮的禁用启用

## 功能概述
多选表格场景下，按钮默认禁用状态，选中多条数据后，遍历判断选中数据的状态是否是指定状态，如果是，解除禁用状态，否则保持禁用。
